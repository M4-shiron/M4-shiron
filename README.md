## Featured Projects

### 🚢 [Titanic Survival Prediction](https://github.com/M4-shiron/Titanic-first-model)
Titanicの乗客データから生存を予測する分類モデルです。

- Random Forestによる二値分類
- 欠損値補完とカテゴリ変数のエンコーディング
- FamilySize（家族人数）の特徴量を作成
- Kaggle Public Score: **0.77990**

### 🚀 [Spaceship Titanic Prediction](https://github.com/M4-shiron/Spaceship-Titanic-model)
乗客が異次元へ転送されたかを予測する分類モデルです。

- Random Forestによる二値分類
- 数値・カテゴリ特徴量の欠損値処理
- Cabinをdeck / sideへ分解する特徴量エンジニアリング
- GridSearchCV（5分割交差検証）: **0.7958**
